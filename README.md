# MyMaintenance Instructions

Tools: MySQL Workbench, MySQL Server, Intellij (Other IDEs are fine) & Postman

Set up a MySQL Server local instance at Port 3306 and ensure it is running, after setting up username and password, please update the application.properties of MyMaintenance project 

Open MyMaintenance project in IDE and run MyMaintenanceApplication

Open Postman and test with provided APIs
